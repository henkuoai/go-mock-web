还没有 mock 接口

点击右上角「新建接口」开始

名称 方法 路径 状态 延迟 启用 操作

支持 :param 路径参数与 * 通配

未设置则默认 Content-Type: application/json

{{.Method}} — 请求方法

{{.Path.id}} — 路径参数（来自 :id）

{{.Query.page}} — 查询参数

{{.Body.name}} / {{.Body.user.id}} — 请求体 JSON 字段

{{.Form.field}} — 表单字段

{{.RawBody}} — 原始请求体

{{index .Header "X-Token"}} — 请求头（键含特殊字符用 index）
