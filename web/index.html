导入 Swagger / OpenAPI

粘贴 Swagger 2.0 或 OpenAPI 3.0 的 JSON。将自动解析每个 path×method 为一条 mock 接口，并按响应 schema 生成示例响应体。路径参数 {id} 会转为 :id。

支持 :param 路径参数与 * 通配

未设置则默认 Content-Type: application/json

{{.Method}} — 请求方法

{{.Path.id}} — 路径参数（来自 :id）

{{.Query.page}} — 查询参数

{{.Body.name}} / {{.Body.user.id}} — 请求体 JSON 字段

{{.Form.field}} — 表单字段

{{.RawBody}} — 原始请求体

{{index .Header "X-Token"}} — 请求头（键含特殊字符用 index）
